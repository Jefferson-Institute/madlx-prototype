## Getting started

### Prerequisites

 * Your machine needs to have the latest [Docker](https://docs.docker.com/engine/) and [Docker compose](https://docs.docker.com/compose/) installed.

### Installing
#### Docker
Run this command from project root
```sh
docker-compose up -d
```
Open http://127.0.0.1 (laravel container port declared in `docker-compose.yml`) to access Dashboard
